# Zenith Nav

极简浏览器导航页，单 HTML 文件，无依赖。

## 功能

- 暗色/亮色主题切换，9 种主题色
- 背景：纯色（60 预设）/ 渐变 / 随机图片 / 壁纸
- 快捷启动：横向图标栏，增删、拖拽排序
- 书签分类：卡片盒子模式，自由拖拽定位 + 调整大小
- Cmd+K 全局搜索
- 侧边栏：暖白色背景，独立对比度控制
- 卡片背景色 / 链接选中色可调
- JSON 导入/导出/重置
- localStorage 数据持久化

## 使用

直接在浏览器中打开 `preview.html` 即可。
