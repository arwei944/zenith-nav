# Zenith Nav — 版本日志

## v5 (当前) — card-boxes
- 书签分类改为卡片盒子模式（自由拖拽定位 + 调整大小）
- 卡片位置/大小持久化到 localStorage
- 添加分类按钮固定在设置按钮旁边
- 侧边栏背景改为暖白色 #f5f2ed
- 侧边栏文字改为深色系（#2c2c2c / #555 / #888）
- 侧边栏对比度滑块基于暖白色调计算
- 新增卡片盒子背景色调节（设置 → 外观）
- 新增链接选中色调节（设置 → 外观）
- 修复链接行 hover 溢出问题（translateX → padding-left）
- 卡片 overflow 改为 clip（保留圆角裁剪，不阻止子元素滚动）

## v4 — sidebar-contrast-fix
- 修复 setInterval 泄漏（init 重复调用时不再创建多个定时器）
- 修复侧边栏对比度滑块无效（CSS 变量引用恢复）
- 修复侧边栏 h2 和 desc 颜色被通用规则覆盖的问题
- 删除纯色模式下重复的对比度滑块
- 侧边栏文字默认色 #f5f2ed，可通过侧边栏对比度滑块动态调节
- 搜索引擎只保留 Google
- 全局间距滑块（控制所有模块间距）
- 侧边栏背景与主背景同步（var(--surface)）
- 侧边栏独立对比度控制（不影响主页）

## v3 — api-wallpaper
- 新增「图片」背景类型，通过 API 获取随机图片
- 支持 3 个分类：美女、动漫、风景
- 「换一张」按钮刷新随机图片
- 背景类型：纯色 / 渐变 / 图片 / 壁纸（4 种）

## v2 — foldable-settings-contrast-bg-presets
- 设置面板改为手风琴折叠式（通用/外观/显示/数据）
- 新增对比度调节滑块（10-100%）
- 背景色预设扩展到 60 个（6 色系 × 10）
- 背景与文字对比度自适应（WCAG luminance）

## v1 — initial
- 初始版本：极简导航页
- 暗色/亮色主题切换
- Cmd+K 全局搜索
- 快捷启动（增删、拖拽排序）
- 分类书签（折叠、编辑、拖拽排序）
- 时间 + 问候语
- 设置面板（站点名、搜索引擎、主题色、背景、显示开关）
- JSON 导入/导出/重置
- localStorage 数据持久化
